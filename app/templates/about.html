About AF3‑DB

AF3‑DB is an open-source, self-hosted web application designed to archive, analyze, and share structure models generated with AlphaFold 3 (© DeepMind). Each entry comes with ipTM / pTM / pLDDT confidence scores, downloadable mmCIF files, biophysical interaction metrics, and rich cross‑links to UniProt / UniParc.

Technical stack
  • FastAPI & SQLAlchemy – REST API, server‑side rendering, and database ORM
  • Bootstrap 5 – layout & styling
  • PDBe Mol* 3.2 – in‑browser molecular viewer
  • Foldseek API & FoldDisco – structural similarity and interface motif search
  • Biotite & Pandas – structural parsing, biophysical property calculations, and data processing

Protein sequences & annotations are retrieved live from UniProt; Foldseek search results are streamed directly from search.foldseek.com.
